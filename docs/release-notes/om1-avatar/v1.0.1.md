## Docker image

The OM1-avatar service is provided as a Docker image for easy setup:
```bash
git clone https://github.com/OpenMind/OM1-avatar.git
```
```bash
cd OM1-avatar
docker-compose up om1_avatar -d --no-build
```
The latest docker image is also available at [Docker Hub](https://hub.docker.com/layers/openmindagi/om1_avatar/v1.0.1).

For more technical details, please refer to the [docs](https://docs.openmind.org/full_autonomy_guidelines/om1_avatar).
